# 🛠️ CLI-Kit (开发者全能工具箱)

CLI-Kit 是一个基于 Python 构建的、**高度插件化**且具备**交互式菜单**的命令行工具集。它旨在通过统一的入口，快速调用各种开发、调试和生产力工具。

---

## 🚀 核心特性

* **全自动扫描**：只需将 `.py` 脚本丢入 `mods/` 文件夹，即可自动在菜单中显示。
* **交互式 UI**：支持键盘上下键选择，智能参数询问，告别死记硬背指令。
* **混合模式**：既支持 UI 菜单操作，也支持命令行参数（CLI）静默执行。
* **一键分发**：支持打包为独立的 `.exe`，且保持插件的动态加载能力。

---

## 📂 目录结构

```text
CLI-Kit/
├── CLI-Kit.exe         # 打包后的主程序
├── mods/               # 插件目录（所有的功能逻辑都在这里）
│   ├── zentick.py      # 专注工具
│   ├── portscan.py     # 诊断工具
│   └── ...
├── core/               # 核心依赖与持久化配置
│   ├── deps.py         # 依赖检查器
│   └── shortcuts.json  # 存储的快捷方式
└── build.py            # 自动化构建脚本
